## Estrategia de A/B Testing Recomendada

**Semana 1-2**: Publicar con copy ACTUAL (A)
**Semana 3**: Swapear a Variante B o C basado en:
- Mejor CTR
- Mejor engagement rate
- Mejor guardados

**Métrica por post**:
```
Engagement Rate = (Likes + Comentarios + Guardados) / Impresiones × 100
Target: >3%
Excelente: >5%
```
